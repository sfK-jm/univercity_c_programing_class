## 실습문제 4: 히스토그램

카운터 배열을 사용해서 히스토그램을 그려라
- 아래와 깉이 15명의 점수가 주어진다
  ~~~c
    int scores[15] = {
        5, 23, 100, 95, 80,
        77, 69, 85, 92, 98,
        54, 62, 90, 76, 85
    };
  ~~~
- 먼저 크기 11인 카운터 배열을 만들고
- 각 점수 구간을 배열 인덱스로 사상한다
- 앞의 카운터 배열을 사용해서 구간별 도수를 계산한 후, 화면에 출력한다
- 도수의 크기만큼 별("*")을 출력해 보자

### 출력 예시
~~~
0 ~ 9: *
10 ~ 19:
...
80~98:***
90~99:****
100: *
~~~
